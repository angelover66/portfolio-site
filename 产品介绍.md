# 产品介绍 — 个人AI作品集网站

> 版本：v1.1 | 日期：2026-05-27

## 一、产品基础定位

**产品名称**：Lulu Yang AI Portfolio

**产品背景**：AI产品经理转型求职过程中，需要一个集中展示个人AI Agent项目的作品集网站。目标访问者为面试官和同行，用于快速了解候选人的产品能力和技术落地能力。

**使用人群**：面试官、同行、潜在合作者。

**核心解决痛点**：
- 5个AI/产品项目分散在不同平台，缺少统一展示入口
- 简历只能写文字，无法展示产品实际运行效果
- 需要传达「AI产品经理 x AI Agent开发者」双重身份

**产品价值**：单页作品集网站，Hero区建立个人品牌认知，卡片网格展示项目，弹窗快速浏览详情，提供在线体验、GitHub和小程序二维码多种访问入口。

## 三、AI技术架构

```
Next.js App Router (SSG)
  ├── app/layout.tsx (根布局 + 字体 + Metadata)
  ├── app/page.tsx (单页主入口，组合全部区块)
  ├── app/components/
  │   ├── Navbar.tsx (顶部导航)
  │   ├── Hero.tsx (首屏)
  │   ├── ProjectCard.tsx (项目卡片)
  │   ├── ProjectModal.tsx (详情弹窗，含二维码/按钮条件渲染)
  │   └── Footer.tsx (页脚)
  ├── data/projects.ts (硬编码项目数据，含Project接口定义)
  └── public/ (静态资源：二维码图片等)
```

**数据模型**：
```typescript
interface Project {
  id: string;
  name: string;
  tagline: string;
  description: string[];
  techTags: string[];
  liveUrl: string;      // 在线链接，小程序为空
  githubUrl: string;    // GitHub链接，未开源为空
  emoji: string;
  qrcode?: string;      // 可选，小程序二维码图片路径
}
```

**技术特点**：纯静态页面，无API调用，无数据库，Next.js SSG生成，Vercel CDN分发。

## 六、异常与边界处理

- 弹窗打开时禁止背景滚动
- ESC键关闭弹窗
- 点击遮罩层关闭弹窗
- 所有外部链接 `target="_blank"` + `rel="noopener noreferrer"`
- 项目无 liveUrl 时隐藏"在线体验"按钮
- 项目无 githubUrl 时隐藏"GitHub 仓库"按钮
- 项目有 qrcode 时显示二维码替代按钮行
- 项目同时无 liveUrl 和 githubUrl 时隐藏整个按钮区域
